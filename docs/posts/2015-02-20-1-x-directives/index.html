1.x Directives

Notes to self & advice to those working in Angular 1.x Land. If you don’t want to read this, here are three elevator pitch points to consider when understanding directives:

  1. Use the compile function to change the original DOM (template element) before AngularJS creates an instance of it and before a scope is created.
    1. Use the pre-link function to implement logic that runs when AngularJS has already compiled the child elements, but before any of the child element’s post-link functions have been called.
    2. Use the post-link function to execute logic, knowing that all child elements have been compiled and all pre-link and post-link functions of child elements have been executed.
