When I write a program i sometime use variables as place holders or counters that will track the progress function that uses looping to do some work for me. Everytime your program goes through the loop you have to tell it to increment the counter variable. If I have a variable named ‘counter’ then I can use a shorthand notation to ‘increment counter.’ This is done by writitng this: counter++. Now you understand the teeth++ above.
